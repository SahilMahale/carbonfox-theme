# Carbonfox Theme for Yaak

A dark, elegant theme for [Yaak](https://yaak.app) based on the popular [Nightfox](https://github.com/keifererikson/vscode-nightfox) VSCode theme, specifically the Carbonfox variant.

## About

This theme is a port of the Carbonfox color scheme from the VSCode Nightfox theme family. It brings the sophisticated dark palette and carefully crafted color contrast to Yaak, providing a comfortable and visually appealing experience for API development.

## Color Palette

Carbonfox features a carefully balanced dark color scheme:

- **Backgrounds**: Deep charcoal (`#161616`) with darker accents (`#0c0c0c`)
- **Text**: Clean, high-contrast grays for optimal readability
- **Accents**:
  - Purple (`#be95ff`) - Primary actions
  - Blue (`#78a9ff`, `#52bdff`) - Info and secondary elements
  - Cyan (`#3ddbd9`) - Success states
  - Orange (`#F4A261`) - Notices and highlights
  - Pink (`#ee5396`) - Errors and warnings

## Installation

1. Clone this repository or download the plugin
2. Install the plugin in Yaak following the [plugin installation guide](https://yaak.app/docs/plugins)
3. Select "Carbonfox" from the theme options in Yaak settings
